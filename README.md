# CodeAlpha_Calculator
This repository contains a basic calculator web application developed using HTML, CSS, and JavaScript. The project demonstrates how to build an interactive and user-friendly calculator that performs essential arithmetic operations.
Features:
Supports basic arithmetic operations: Addition (+), Subtraction (−), Multiplication (×), Division (÷)
Interactive button-based user interface
Real-time input and result display
Clear (C) and delete (⌫) functionality
Technologies Used:
HTML – Structure of the calculator
CSS – Styling and layout
JavaScript – Logic, calculations, and event handling
